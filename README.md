# ELEVATE Project Page

这是一个可直接用于 GitHub Pages 的科研项目展示主页，已将当前目录中的 PDF 材料接入页面：

- `ELEVATEEvent-triggered Low-altitude Enhanced.pdf`
- `pipeline.pdf`
- `VQApipeline.pdf`
- `qualitative_results5.pdf`
- `qualitative_results6.pdf`

## 本地文件

- `index.html`：主页内容
- `styles.css`：页面样式

## 发布到 GitHub Pages

1. 在 GitHub 新建仓库。
2. 将当前目录文件上传到仓库根目录。
3. 打开 GitHub 仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` / `/ (root)`
5. 保存后等待几分钟，GitHub 会生成项目主页链接。
